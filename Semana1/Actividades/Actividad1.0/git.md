# Actividad 1.0: Configuración de Git y Repositorio

## Descripción
Configuración del entorno de trabajo con Git y GitHub.

## Pasos realizados
1. Cuenta creada en GitHub
2. Repositorio creado: CienciaDeDatos
3. Estructura de carpetas creada
4. Repositorio inicializado con git init
5. README.md creado
6. Repositorio conectado con git remote
7. Primer commit realizado

## Comandos utilizados
git init
git config --global user.name "Mi Nombre"
git config --global user.email "mi@email.com"
git remote add origin https://github.com/mi_usuario/CienciaDeDatos.git
git add .
git commit -m "Initial commit"
git push -u origin main

## Entregable
Repositorio creado y funcionando correctamente.
